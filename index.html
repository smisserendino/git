Áreas de GIT

Working Directory

Es el área de trabajo, donde vamos a crear, editar, elimnar los diferentesa archivos que formeen parte de nuestro proyecto.

Staging Area (INDEX)

Es el área de control de cambios, donde se realizan las capturas de codigo snapshot y basicamente donde indicamos a que archivo vamos estar realzando seguimiento.

Repository

Es el area donde se almacenan las confirmaciones de los cambios realizados en archivos de tipo BLOB (Binary Large Object File), se utilizan como puntos de referencia a los que podemos volver siempre y cuando indiquemos a git cual es el numero que HASH o posicion que queremos visualizar.
